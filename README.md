# AI献立コンシェルジュ (AI Menu Concierge)

### 概要
本プロジェクトは、LLMアプリケーション開発プラットフォーム「Dify」と、iPaaSツールの「Make」を組み合わせ、日常の献立決定をサポートするLINEボットです。

### 技術スタック
* **LLM Platform**: Dify
* **iPaaS**: Make
* **Interface**: LINE Messaging API

### 開発工程
1. **DifyによるAIロジックの構築**: 3日間の献立作成、買い物リスト、使い回しポイントを生成するプロンプトエンジニアリングを実施。
2. **LINE Messaging APIの設定**: Webhookを用いたリアルタイム通信基盤の構築。
3. **Makeによるオーケストレーション**: LINE(受信) → Dify(推論) → LINE(返信) のフローをノーコードで実装。

### 工夫した点
* 調理負荷を減らすための「ネガティブ制約（揚げ物禁止など）」の設計。
* 1つの食材を3日間で使い切るための在庫最適化ロジックの組み込み。
* セキュリティを考慮し、APIキー等の機密情報を除外した設計図の公開。
